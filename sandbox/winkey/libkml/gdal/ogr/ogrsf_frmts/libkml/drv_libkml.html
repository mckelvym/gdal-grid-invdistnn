Schema

fields

LIBKML_NAME_FIELD
Name of the OGR field to use for a placemarks name, Defaults to "Name"
LIBKML_DESCRIPTION_FIELD
Name of the OGR field to use for a placemarks description, Defaults to Description
LIBKML_TIMESTAMP_FIELD
Name of the OGR field to use for a placemarks timestamp, Defaults to timestamp
LIBKML_BEGIN_FIELD
Name of the OGR field to use for a placemarks timespan begin, Defaults to begin
LIBKML_END_FIELD
Name of the OGR field to use for a placemarks timespan end, Defaults to end
